# Qt 摄像头实时采集与抓拍系统

本项目基于 Qt 框架开发，实现了高性能的摄像头实时画面预览与图像抓拍功能。项目针对特定的硬件采集卡或工业相机进行了配置优化，支持 YUY2 原始流处理。

## 🚀 项目特性

* **实时预览**：支持 30FPS 流畅预览，分辨率锁定为 **640 * 360**。
* **硬件加速**：集成 `XDMA_MoreB` 驱动库，优化数据传输效率。
* **格式支持**：直接处理 **YUY2** 像素格式图像。
* **一键抓拍**：支持实时画面截取并保存为本地图片。
* **跨编译器支持**：完美兼容 MSVC 和 MinGW 编译器，统一采用 UTF-8 编码避免中文乱码。

## 🛠 技术规格

| 参数项 | 设定值 | 说明 |
| :--- | :--- | :--- |
| **分辨率** | 640 * 360 | 默认工作分辨率 |
| **帧率 (FPS)** | 30 | 实时采集频率 |
| **像素格式** | YUY2 | 摄像头/采集卡原始输出 |
| **C++ 标准** | C++17 | 使用现代 C++ 特性提升代码质量 |
| **核心模块** | Multimedia | 使用 `multimediawidgets` 渲染 |

## 📦 环境要求

* **Qt 版本**: Qt 5.12.10 及以上 (兼容 Qt 6)
* **编译器**: MSVC 2017 64bit
* **依赖项**: 
    * 需要加载 `driver/` 目录下的 `XDMA_MoreB` 库（仅限 win32 环境）。
    * 项目已预设 UTF-8 源码字符集。
